# Развертывание клиент-серверного приложения локально

# Развертывание сервера

1. Создайте локальную базу данных MySQL на порту 3306 с названием tamaco_db, создайте пользователя со слудующими данными name: root, password rootroot

2. В папке проекта backend создайте файл .env

3. Скопируйте и вставьте в файл .env следующий код (если вы создали базу данных на другом порту, с другим именем или пользователем, замените в коде соответствующие данные на Ваши):

# БД
DB_HOST=localhost
DB_PORT=3306
DB_NAME=tamaco_db
DB_USER=root
DB_PASSWORD=rootroot
# Порт сервера
PORT=3001
# Настройки JWT
JWT_SECRET=ваш_секретный_ключ_минимум_32_символа
JWT_EXPIRES_IN=24h
# Безопасность
BCRYPT_SALT_ROUNDS=10

4. Откройте терминал, перейдите в папку проекта tamaco/backend и запустите код npm run dev
Сервер запустится на порту 3001 (если вы меняли порт сервера в файле .env, то сервер запустится на указанном вами порту)

5. Для проверки работоспособности сервера откройте в браузеер страницу:
http://localhost:3001/api/health
Должна отобразится информация об успешном запуске Tamaco API

# Развертывание клиентского приложения

1. В папке приложения /frontend созайте файл .env.development

2. Скопируйте и вставьте в файл .env.development следующий код (если вы меняли порт сервера на шагах выше,замените порт 3001 на свой порт):

REACT_APP_API_URL=http://localhost:3001/api
REACT_APP_NAME=Tamaco
REACT_APP_VERSION=1.0.0

3. Откройте НОВЫЙ терминал, перейдите в папку проекта tamaco/frontend и запустите код npm start
Сервер запустится на порту 3000

4. Перейдите в браузере по адресу для тестирования приложения http://localhost:3000/
